## Notes (React Hooks)


### Installation / Development

```bash
    # clone project
    git clone <repo>
    cd project name
    
    # install dependecies
    npm install
    
    # start server
    npm start
```
Visit: http://localhost:3000

### Build / Production

```bash
    npm install gh-pages --save-dev
    npm run deploy
```

### Includes

- react v.16
- bootstrap(used sass)
- node-sass

### Features

* Display all notes
* Add note and Save note as drafts
* Delete note
* Sort notes by date
* Filter (search) notes by title
* Focus input field (add note form)
